## 01 Recursion & Backtracking
- Basic concepts of recursion.
- Solving problems like N-Queens, Rat in a Maze, Permutations, and Subsets using recursion and backtracking.

## 02 Dynamic Programming (DP)
- Introduction to DP, Top-Down and Bottom-Up approaches.
- Problems like Fibonacci, Knapsack, Longest Common Subsequence, and more.

## 03 Greedy Algorithms
- Explanation of greedy strategies and optimization.
- Includes problems like Activity Selection, Huffman Encoding, and Fractional Knapsack.

## 04 Graphs
- Graph traversal algorithms such as BFS, DFS.
- Solving problems related to shortest paths, spanning trees, and connected components.

## 05 Searching
- Linear and Binary search algorithms.
- Problems like Search in Rotated Sorted Array, Searching in 2D Matrix.

## 06 Sorting
- Various sorting algorithms including Merge Sort, Quick Sort, Bubble Sort, and others.
- Explanation of time complexity and sorting problem solutions.

## 07 Pattern Searching
- Algorithms to search for a pattern in a string such as KMP (Knuth-Morris-Pratt), Rabin Karp, and others.

## 08 Bitwise Algorithms
- Bit manipulation techniques.
- Solving problems like finding the single non-repeated element, counting set bits, etc.

## 09 Mathematics
- Mathematical algorithms and techniques for solving number theory problems.
- Prime factorization, GCD, LCM, and more.

## 10 Trie
- Introduction to Tries for efficient searching in a set of strings.
- Solving problems like autocomplete, word search, etc.

## 11 Segment Tree
- Segment tree data structure for range queries.
- Use cases such as range sum, range minimum, and range maximum queries.

## 12 Fenwick Tree
- Binary Indexed Tree (BIT) for efficient prefix sum and range queries.

## 13 Geometric Algorithms
- Algorithms to solve problems related to geometry.
- Convex Hull, Line Intersections, etc.

## 14 Binary Tree and Binary Search Tree (BST)
- Explanation of binary tree, binary search tree, and tree traversal techniques.
- Problems like Inorder, Preorder, Postorder traversals, and BST operations.

## 15 Linked List
- Operations and implementations of singly linked lists, doubly linked lists, and circular linked lists.
- Common problems like reversal, cycle detection, and merging lists.

## 16 Hashing
- Understanding hash tables and hash maps.
- Problems related to hash collision handling, frequency counting, etc.

## 17 Two Pointer and Sliding Window
- Techniques for solving array and string problems efficiently using the two-pointer approach.
- Problems like subarrays with given sum, longest substring with distinct characters.

## 18 Stack, Queue, and Heap
- Stack, queue, and priority queue (heap) implementations.
- Solving problems like next greater element, evaluating expressions, and more.

## 19 Prefix Sum
- Prefix sum technique for optimized range queries.
- Use cases like solving range sum problems.

## 20 Other Advanced Data Structures and Algorithms
- Includes advanced data structures like Suffix Array, Suffix Tree, etc.
- Advanced algorithms like Ford-Fulkerson, Kahn's Algorithm for Topological Sort.

## 21 Randomized Algorithms
- Algorithms that use randomness to achieve better performance.
- Problems like Randomized QuickSort, Monte Carlo methods, etc.

## 22 Combinatorics
- Combinatorics problems including Permutations, Combinations, and Pigeonhole Principle.

## 23 Array and Matrix and String
- Array manipulation, matrix traversal, and string operations.
- Problems like rotating arrays, spiral matrix traversal, and pattern matching.

## 24 Divide and Conquer
- Divide and conquer approach for breaking problems into smaller sub-problems.
- Solving problems like Merge Sort, Quick Sort, and Binary Search.
